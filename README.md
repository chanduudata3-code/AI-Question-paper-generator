# AI Question Paper Generator

An AI-powered Question Paper Generator built using Flask and Ollama (local LLM).

## 🚀 Features

- Generate questions from a topic
- Upload PDF and generate questions
- Select difficulty levels (Easy, Medium, Hard)
- Generates structured JSON
- Exports as downloadable PDF
- Modern UI

## 🛠 Tech Stack

- Python
- Flask
- Ollama (LLaMA3)
- HTML/CSS
- ReportLab (PDF Generation)

## 📦 Installation

1. Clone repository
2. Install dependencies:

pip install -r requirements.txt

3. Run Ollama:

ollama serve

4. Run project:

python run.py
